500 - n. An error message indicating that an unspecified server error has occurred. It is one of a variety of 5xx status codes, all of which describe errors resulting from a problem with the server. Other errors include:
  • 501Not Implemented
  • 502Bad Gateway
  • 503Service Unavailable
  • 504Gateway Timeout
  • 508Loop Detected
